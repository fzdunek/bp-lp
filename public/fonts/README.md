# Dejanire Sans Font Files

Umieść tutaj pliki czcionki Dejanire Sans w następujących formatach:

## Wymagane pliki:

- `DejanireSans-Regular.woff2` (lub .woff, .ttf)
- `DejanireSans-Medium.woff2` (lub .woff, .ttf)
- `DejanireSans-SemiBold.woff2` (lub .woff, .ttf)
- `DejanireSans-Bold.woff2` (lub .woff, .ttf)

## Format plików:

Preferowane formaty (w kolejności):
1. `.woff2` - najlepsza kompresja i wydajność
2. `.woff` - dobra kompresja
3. `.ttf` - uniwersalny format

## Źródło:

Dejanire Sans można zakupić na: https://www.re-type.com/fonts/dejanire-sans/

## Uwaga:

Po dodaniu plików czcionki, upewnij się, że nazwy plików w `resources/css/app.css` odpowiadają rzeczywistym nazwom plików.
